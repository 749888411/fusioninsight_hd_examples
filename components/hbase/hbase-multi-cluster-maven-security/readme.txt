场景描述：
1.集群1为C30L版本，集群2为C50版本；这两个集群做了互信。
2.本DEMO需访问集群上的CTBase和HBase组件，同时访问集群2上的HDFS组件；

操作：
1.在集群1上创建tester1用户，让其具备操作集群1的HBase和CTBase；
2.在集群2上创建同名用户tester1，让其具备操作集群2的HDFS组件；
3.下载集群1客户端，建立HBase组件代码样例工程；
4.下载集群2的HDFS客户端，在3的HBase组件代码工程中创建“conf2”路径，将HDFS的配置信息放在此路径下；创建“lib-hdfs”路径，将HDFS的依赖包放在此路径下。
5.参考本DEMO样例，优化3的代码样例工程。
